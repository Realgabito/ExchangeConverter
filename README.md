# Exchange Converter

## Descripción
**Exchange Converter** es un conversor de monedas desarrollado en Java que permite realizar cálculos entre diferentes monedas. Este proyecto está diseñado para ser simple y eficiente, proporcionando conversiones precisas entre monedas.

## Requisitos
Este proyecto no requiere ninguna dependencia externa. Solo necesitas clonar el repositorio y ejecutarlo en tu entorno Java.

## Tecnologias 
Java
API: https://www.exchangerate-api.com/
Gson
Git

## Instalación

Sigue estos pasos para instalar y ejecutar el proyecto:

1. Clona este repositorio en tu máquina local:
    ```bash
    git clone https://github.com/usuario/exchange-converter.git
    ```

2. Abre el proyecto en tu entorno de desarrollo Java preferido (Eclipse, IntelliJ, etc.).

3. Ejecuta la aplicación.

## Uso

Una vez que la aplicación esté en funcionamiento, simplemente sigue las instrucciones que aparecerán en la consola para realizar conversiones entre monedas.
